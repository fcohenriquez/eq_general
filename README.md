# eq_general
Modelo de equilibrio general para evaluar impuestos

En esta rama se va a evaluar un modelo de 3 sectores con sector externo.


El modelo que se utiliza es gEcon montado en R. En este link está su documentación.

https://gecon.r-forge.r-project.org/

En la actualidad se está desarrollando la estimacion en los archivos gecon_chile_3_sect_comp.Rmd y cge_calib_chile_3_sect_comp.gnc
